# RootsCalculator

A simple Java App that displays the roots of a quadratic equation

# Requirements:
  1. At least Java JRE 8: http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

# How to run:
 ### If on windows:
 1. Execute the Exe file
 2. Golden!
 ### If on anything else:
 1. Execute Jar file
 2. Golden!
